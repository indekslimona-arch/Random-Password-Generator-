# Random Password Generator

## Автор
Алексвндр Кудрявцев

## Описание
«Random Password Generator» — это графическое приложение на Python, позволяющее пользователям генерировать случайные пароли с настраиваемыми параметрами. Пользователи могут выбирать длину пароля, а также типы символов, которые следует включить в пароль (буквы, цифры и специальные символы). Приложение также сохраняет историю сгенерированных паролей в формате JSON.

## Инструкция по запуску
1. **Установите Python** (если еще не установлен):
   - Перейдите на [python.org](https://www.python.org/downloads/) и скачайте последнюю версию Python.

2. **Установите необходимые библиотеки**:
   Убедитесь, что у вас установлены библиотеки `tkinter` (обычно входит в стандартную библиотеку Python), `random`, `string`, `json`.

3. **Скачать проект**:
   - Клонируйте или скачайте репозиторий с GitHub:
     ```bash
     git clone https://github.com/ваш-логин/weather-diary.git
     ```

4. **Запустите приложение**:
   - Перейдите в папку приложения и выполните:
     ```bash
     python password_generator.py
     ```

## Примеры использования
1. **Генерация пароля**:
   - Установите длину пароля с помощью ползунка (например, 10).
   - Выберите типы символов (буквы и цифры).
   - Нажмите кнопку «Сгенерировать».
   - Вы получите сгенерированный пароль, который будет отображен на экране.

2. **Сохранение истории**:
   - После генерации пароля он будет автоматически сохранен в истории.
   - Вы можете просмотреть историю сгенерированных паролей в таблице.
